Forms (Bootstrap 4.6)

The form-control class goes on the input itself. it styles the input to look much nicer.

The form-group "encourages proper grouping" of form components. It doesn't do much.

There are also classes for sizing; form-control-lg and form-control-sm.


For checkboxes, there are two classes; form-check-input and form-check-label. Bootstrap comes with its own checkbox styles as well (refer to the documentation on how to use those because it is very different between 4.6 and 5.3 which is current at the time of writing).

Form Layouts

To layout the form, the grid method (with the column classes) can be used.
